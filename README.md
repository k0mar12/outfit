## Quick Setup

1. Add `nuxt-outfit` dependency to your project

```bash
# Using npm
npm install --save-dev nuxt-outfit
```

2. Add `nuxt-outfit` to the `nuxt-outfit` section of `nuxt.config.ts`

```js
export default defineNuxtConfig({
  modules: [
    'nuxt-outfit'
  ]
```
